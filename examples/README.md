# Examples

완성된 데모들을 모아두는 곳입니다. 각 예제는 자기 폴더 안에 소스, 사용한 에셋,
렌더링된 최종 영상까지 전부 들고 있어서(자기완결적) 그대로 열어보고 재생할 수 있습니다.

`manim/scenes/`, `motion-canvas/src/scenes/`는 작업 중인/기본 템플릿 씬을 위한 곳이고,
여기 `examples/`는 "완성해서 남겨두는" 결과물 전용 폴더입니다. 새 예제를 만들 때마다
`examples/<example-name>/` 폴더를 하나 새로 만들어 같은 패턴으로 정리해주세요.
